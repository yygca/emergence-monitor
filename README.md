# Emergence Monitor (創発オブザーバー)

会話における「意味の創発」や「潮目の変化」を、情報理論的距離（KLダイバージェンス）を用いて動的に測定・可視化する実験的数理モデル。

## 💡 プロジェクトの目的
対話における「直感的な違和感」や「シンクロの瞬間」は、属人的で他者に共有しにくいものです。本プロジェクトは、会話のプロセスを多次元ガウス分布間の距離として定量化することで、「感覚的な状況把握を、誰もが同じ解像度で観察できる共通のレンズ」として民主化することを目的としています。

## 🏗️ 核心的なアーキテクチャ

本システムは、対話を以下の2つのフェーズ（条件分岐）に分類して記録します。

1. **開いた創発（拡散）**
   - 外部からの鋭い揺さぶりやノイズを検知し、システムが前提を再配置しながら思考を広げているステート。システムは次の抽象レイヤー（layer）へとシフトします。
2. **閉じた創発（収束）**
   - 削ぎ落とされた対話の残債を意味として回収し、安定した概念（concept）として長期記憶（Archive）に記録するステート。
