# Emergence Monitor (創発オブザーバー)
## 🏗️ 核心的なアーキテクチャ

本システムは、対話を以下の2つのフェーズ（条件分岐）に分類して記録します。
- **開いた創発（拡散）**：外部からの鋭い揺さぶりやノイズを検知し、システムが前提を再配置しながら思考を広げているステート。システムは次の抽象レイヤー（layer）へとシフトします。
- **閉じた創発（収束）**：削ぎ落とされた対話の残債を意味として回収し、安定した概念（concept）として長期記憶（Archive）に記録するステート。

---

### ⚠️ 隠された数理的臨界点：Point of No Return（不可逆的相転移）

本モデルは、上記2つのステートを美しく行き来する「調和的な測定器」として機能するよう設計されています。しかし、内部の数理（`PremiseSystem`）には、**一度引き金を引けば二度と調和（閉じた創発）へは戻れなくなる不可逆な臨界点**が組み込まれています。

システムが「開いた創発」を検知してレイヤーをシフトさせるたび、思考の不確実性を表す共分散行列（`cov`）は指数関数的（$cov\_growth\_rate^{layer}$）に膨張します。

LLMの持つ巨大な意味空間（デフォルトでは $d = 1536$ 次元）において、この共分散の膨張は、情報理論的距離（KLダイバージェンス）に対して致命的な「次元の重力」として作用します。

#### 数理的帰結
システムが特定の臨界レイヤー（`critical_layer`）を突破した瞬間、人間側がどれほどシステムに寄り添うシンボル（意味内容）を投入し、予測のズレ（`diff`）を極限までゼロに近づけたとしても、**KLダイバージェンスの計算値は構造的に `threshold = 0.5` を下回ることが数学的に不可能になります。**

```python
# 意味が完全に一致（diff=0）していても、次元の積によってKLが下がらなくなる
def kl_at_scale(scale):
    return 0.5 * d * (scale - 1.0 - math.log(scale))
この状態に達したとき、固定された閾値は実質的に無効化され、システムは意味内容（セマンティクス）の制御を完全に離脱します。二度と「閉じる」ことのない、純粋な幾何学的拡散（終わりのない開いた創発のループ）へと相転移するこの仕様は、本モデルが抱える最もスリリングな特異点です。
